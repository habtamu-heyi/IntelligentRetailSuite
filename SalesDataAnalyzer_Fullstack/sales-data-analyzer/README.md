## Description
A fullstack app to upload sales CSV data and instantly get key insights like:
- Total revenue
- Top & worst performing products
- Average sale amount

## Stack
- Flask (API)
- React (UI)
- Axios (Requests)
- Pandas (Analysis)

## Run Backend
```bash
cd backend
pip install -r requirements.txt
python app.py
```

## Run Frontend
```bash
cd frontend
npm install
npm start
```
