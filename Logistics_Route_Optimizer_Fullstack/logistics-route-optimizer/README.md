
# ğŸšš Logistics Route Optimizer (Fullstack App)

## Overview
A web app that optimizes delivery routes using a simple backend algorithm and a React-based interface.

## Getting Started

### Backend
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## Example Input:
```
Warehouse, Store A, Store B, Store C
```

## Example Output:
```
Store C -> Store B -> Store A -> Warehouse
```
